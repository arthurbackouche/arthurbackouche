How to create a Data Extension that contains unsubscribers

The objective of this feature is to create a Data Extension that contains information related to the users that are unsubscribing from Marketing Cloud. We will use the Data View: Unsubscribe that allows us to pull these information. Add the following field to the Data Extension:

Create a Target Data Extension​

We need to store somewhere the information related to users that are unsubscribing from Marketing Cloud. We will create a Data Extension with the name MasterUnsubscribers.

[SubscriberKey][Text + Primary Key]
[EventDate][Text]
[JobID][Text]
[ListID][Text]
[BatchID][Text]

Create an SQL Query in Automation Studio​

The first step is to create an SQL Query in Automation Studio. From the Marketing Cloud home page we are going in Journey Builder and then Automation Studio. From there we can click on the button New Automation and then we will Drag&Drop the widget Schedule as an Starting Source. We want this Automation to be refreshed on a hourly basis and it should never end. Next, we can Drag&Drop the SQL Query widget and click Choose and then Create New Query Activty.

After Giving generic information to the Query such as Name, External Key, Folder Location and Description.

Please find below the code of the Query:

SELECT
SubscriberKey AS SubscriberKey,
EventDate AS EventDate,
JobID AS JobID,
ListID AS ListID,
BatchID AS BatchID
FROM _Unsubscribe

You can now save the Automation. Don't forget to select Overwrite as the way of pushing the data in the Target Data Extension. Finally you can save the Automation give it a Name, a Description and an External Key and click Save.

Test and Activate the Automation​

Once you saved your Automation, you can click on Run Once, then click Select All Activities and Run. Now go in the Activity tab of your Automation. You should see the test running with the SQL Query widget in Blue.

If the SQL Query is blue the test is running. If the SQL Query is red the test is errored. If the SQL Query is green the test is complete.

Once the test is complete, you can go in your Target Data Extension MasterUnsubscribers and check the results.
